# Example results

These files are generated by the comprehensive examples in `examples/`.

They are useful for users who want to see expected output shapes before running
the scripts themselves:

- `05_*`: static panel OLS, pooled OLS, FE, RE, IV/2SLS, and post-estimation
- `06_*`: Difference GMM and System GMM coefficients, diagnostics, and model card
- `07_*`: panel CV, forecasting, backtesting, and diagnostic-first GMM search
- `08_*`: post-estimation summaries and diagnostic figures

These are not formal validation artifacts. External-software parity and review
evidence are kept under `artifacts/`.
