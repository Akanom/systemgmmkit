# Accepted Risks

No dependency or scanner finding is currently accepted as a release exception.

The NumPy 2.5.1 opaque-archive finding remains `MONITOR`, not `ACCEPTED RISK`,
until the scanner artifact hash is available and matches official PyPI
metadata. Future entries must identify an owner, exact package version,
artifact SHA-256, alert, evidence, scope, review date, expiry or reassessment
trigger, and compensating controls.
